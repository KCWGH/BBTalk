# BBTalk (BlackBerry 9900 KakaoTalk Web UI)

블랙베리 9900(Bold) 기종의 브라우저 환경에 최적화된 카카오톡 웹 클라이언트입니다. 안드로이드 기기에서 수집된 알림을 수신하고, 블랙베리에서 답장을 보낼 수 있는 중계 서버 역할을 수행합니다.

## 핵심 기능

* **BB 9900 맞춤형 UI**: 좁은 화면과 트랙패드 조작에 최적화된 인터페이스.
* **상단 토글 라인 대응**: 블랙베리 OS 브라우저의 주소창 토글 기능을 고려한 상단 여백 설계.
* **실시간 메시지 동기화**: `asyncio` 이벤트를 통한 실시간 대화 상태 업데이트.
* **프로필 이미지 지원**: Base64 기반의 프로필 이미지 렌더링.
* **중복 메시지 방지**: 고유 메시지 ID(`msg_id`) 검증 로직 탑재.

## 프로젝트 구조

* `main.py`: FastAPI 기반의 백엔드 서버 로직.
* `templates/`: Jinja2 템플릿 엔진을 사용한 채팅 목록 및 대화창 HTML.
* `static/`: CSS, 폰트 및 파비콘 등 정적 리소스.
* `requirements.txt`: 프로젝트 실행에 필요한 의존성 라이브러리 목록.
